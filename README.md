# Godot-Modules-by-Tony
A bunch of useful classes and tools ready to be used inside your Godot projects!
For Godot 4 and other random versions I guess xD.

Includes, for example:
- A free LLM API implementation ready to be used inside Godot!
- An improved version of the HBoxContainer and VBoxContainer for when you need a little more control.
- And other stuff.

IMPORTANT:
Some things may not work or be unfinished.
Treat this more like an attic with a bunch of random stuff your grandpa as been crafting over the years, some that might not be finished.
